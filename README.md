# Face_Emotion_Detection_System
Face Emotion Detection System is a Python-based computer vision and deep learning project that detects human emotions from facial expressions in real time or from images/videos. The system uses OpenCV for face detection and a trained deep learning model to classify emotions such as Happy, Sad, Angry, Surprise, Neutral, Fear, and Disgust.
